

## RASI  Game Platform

A comprehensive lottery betting platform where users can place bets on lottery results based on digit combinations, with UPI payments and full admin management.

---

### üéØ Core User Features

**1. User Authentication & Onboarding**
- Sign up and login with email/password
- Profile management with basic user info
- Wallet balance display throughout the app

**2. Lottery Selection & Game Board**
- Browse available lotteries (Kerala Lottery, and others you'll add later)
- View lottery draw time and countdown timer
- See previous winning numbers for reference

**3. Betting Interface (as shown in your screenshots)**
- **Single Digit Bets**: Choose position A, B, or C with custom number
- **Double Digit Bets**: Choose combinations AB, BC, or AC
- **Three Digit Bets**: Enter full ABC combination with BOX option
- Quantity selector for multiple tickets
- "Quick Guess" random number generator
- Shopping cart showing selected bets and total amount

**4. Wallet & Payments**
- View wallet balance and transaction history
- Add funds via UPI payment gateway
- Withdraw winnings to bank account (admin approved)
- Transaction receipts and history

**5. Bet History & Results**
- View all placed bets with status (pending/won/lost)
- See winning amount calculations
- Filter by lottery, date, and bet type

---

### üëë Admin Dashboard

**1. Lottery Management**
- Create/edit lottery types (name, timing, ticket prices, win amounts)
- Set draw schedules and active/inactive status

**2. Results Entry**
- Enter winning 3-digit number after each draw
- System automatically calculates winners across all bet types
- View result history

**3. User Management**
- View all registered users
- See user wallet balances and bet history
- Manage user accounts (activate/deactivate)

**4. Financial Reports**
- Daily/weekly/monthly betting summaries
- Total bets placed vs winnings paid
- Wallet top-up and withdrawal tracking
- Profit/loss overview

**5. Payment Approvals**
- Approve/reject deposit confirmations
- Process withdrawal requests
